# TypeScript-Node-Template

Quickly starting to build a project using TS in Node.

## Usage

### Development

```bash
npm run dev
```

It will enter watch-mode with `tsc`

### Production

```bash
npm run build
```

It uses [rollup](https://github.com/rollup/rollup) for building production codes.
